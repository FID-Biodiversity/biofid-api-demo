# BIOfid-API examples

In the Jupyter notebooks, you will find some code examples of how to analyse data retrieved from the BIOfid
semantic search portal. Remember: This is no publication ready data. Checking the respective
sources given in the data is mandatory!

If you have more ideas or have any suggestions on how to improve and/or enrich the returned data, please do not
hesitate to contact us: https://www.biofid.de .
